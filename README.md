Aplicativo de Monitoramento de Vídeos com a função de Detecção de Movimentos.

Linguagem Python

Bibliotecas: 
  
  opencv_contrib_python (cv2)
  
  Matplotlib
  
  Numpy
  
  PyQt6

O aplicativo faz a detecção de movimentos marcando o perímetro em tempo real do que esta se movimentando. Além, disso uma tela abaixo mostra o varrimento de pixels também em tempo real para fins de apresentação educacional.

O aplicativo permite a entrada de vídeo ao vivo com como webcam como vídeos gravados em arquivos.
